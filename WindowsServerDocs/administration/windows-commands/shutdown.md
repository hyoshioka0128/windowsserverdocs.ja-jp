# <a name="shutdown"></a>shutdown



一度に 1 つのローカル コンピューターまたはリモート コンピューターをシャットダウンまたは再起動できます。

このコマンドを使用する方法の例については、[例](#BKMK_examples)を参照してください。

## <a name="syntax"></a>構文

```
shutdown [/i | /l | /s | /r | /a | /p | /h | /e] [/f] [/m \\<ComputerName>] [/t <XXX>] [/d [p|u:]<XX>:<YY> [/c "comment"]] 
```

## <a name="BKMK_examples"></a>例

アプリケーションを強制的に終了して、"アプリケーション:" という理由で1分の遅延後にローカルコンピューターを再起動するには、次のようにします。メンテナンス (計画済み) "および" myapp の再構成 "のコメント
```
shutdown /r /t 60 /c "Reconfiguring myapp.exe" /f /d p:4:1
```
同じパラメーターを使用してリモートコンピューター \\ @ no__t-1ServerName を再起動するには、次のように入力します。
```
shutdown /r /m \\servername /t 60 /c "Reconfiguring myapp.exe" /f /d p:4:1
```
