# <a name="bitsadmin-gethttpmethod"></a>bitsadmin gethttpmethod

使用する HTTP 動詞を取得します。

## <a name="syntax"></a>構文

```
bitsadmin /GetHttpMethod <Job>
```

## <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------|-----------|
|Job|ジョブの表示名または GUID|
