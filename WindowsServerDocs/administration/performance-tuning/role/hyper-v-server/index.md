# <a name="performance-tuning-hyper-v-servers"></a>Hyper-V サーバーのパフォーマンス チューニング


Hyper-V は、Windows Server 2016 での仮想化サーバーの役割です。 仮想化サーバーは複数の仮想マシンをホストできます。これらの仮想マシンは、プロセッサ、メモリ、および I/O デバイスを仮想化することで、相互に分離されていながら、基盤となるハードウェア リソースを共有できます。 仮想化では、サーバーを 1 つのマシンに統合することで、リソース使用量とエネルギー効率を向上し、サーバーの運用コストと保守コストを削減できます。 さらに、仮想マシンと管理 API によって、リソースの管理、負荷分散、およびシステムのプロビジョニングをより柔軟に行うことができます。
